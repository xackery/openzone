     EasyTable, version 1.20. View detailed description

EasyTable is the Delphi component providing access to a database in its own format in fast and easy way. It is a TTable replacement supporting most of its features such as master/detail and filtering, and some special ones such as data encryption, BLOB data compression and multi-indexes. EasyTable does not use BDE, has small footprint, provides fast access and has the fastest search. It is fully compatible with standard DB-aware visual controls, allows using standard database operations and supports calculated and lookup fields. EasyTable provides data importing from any datasource. EasyTable has an excellent search performance. Now TEasyTable has the fastest searching engine. It is even faster than the most of $300-$500 engines such as DBISAM.
EasyTable is now available in the following three variations:
- EasyTable Free is a free variant for non-commercial use only;
- EasyTable Com is a general purpose variant with no restrictions for $30 only;
- EasyTable Pro contains full source code in addition and costs $70.
