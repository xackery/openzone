1. Это просто кнопка (настоящая; наследница TButton)
   Краткое описание свойств:
        14 форм кнопок по умолчанию + Пользовательская форма;
        7 стилей границы кнопки;
        3 стиля градиента;
          Введено определение градиента по Bitmap'у;
        3 стиля Caption;
        Регион фокуса может быть отключен, включено авторское рисование (custom painting)
        Сама кнопка, Caption и градиент могут быть повёрнуты на произвольный угол, даже для пользователь
               ской формы
        Просчет света тени в реальном времени
        3 вида анимации (на вход, выход и нажатие кнопки)
        Точка вращения кнопки может быть указана
        Кнопка может вращаться в реальном времени, даже во время проектирования.(Не используя таймер)
        Все составляющие кнопки могут быть произвольно размещены на её поверхности.
        Все формы кнопки независимо масштабируются по осям X и Y 
  RgnEditor Редактор регионов (Форм кнопок).
  Он обеспечивает следующее:
        Определение региона по Bitmap
        Это определение доступно как по Фоновому цвету так и цвету Переднего плана
        Цвета могут быть определены приблизительно (например можно выделить регион из сосканированного образа)
        Это также пример использования IAeverButton.
  RotateRgn1 это Unit обратого инжиниринга из внутреннего представления регионов в виде прямоугольников
 в Поли-полиномы с автоматическим определением степени их вложенности, а также расчёта границ кнопок.

3. RGNeditor редактор регионов, который может выделить требуемый регион из пользовательского Bitmap
  Чтобы сделать это:
  3.1. Нарисуйте картинку в любом редакторе (например сосканируйте или нарисуйте вручную) и сохраните с расширением BMP .
  3.2. Загрузите Bitmap в RGNEditor(Автоматический пересчет региона в реальном времени можно запретить(реализовано в параллельном процессе))
  3.3. Выберите цвет Переднего или Заднего фона (нажмите на соответствующую кнопку и выберите цвет с рисунка)
  3.4. Отдельные точки можно ликвидировать установив значение Smooth в верхней части редактора
  3.5. Установив величину Accuracy можно расширить определение выбранного цвета(все цвета вблизи цвета заднего фона, например,
                                    будут определены как цвет заднего фона)
  3.6. Нажмите кнопку Ok
  Не забудьте установить стиль кнопки в bkUser!

4. RotateRgn1 это unit с функциями обратного инжиниринга внутреннего представления HRGN в вид PolyPoly(Поли-полином) .
 
Функция                      Действие

RotateRegion                 Вращает регион на определённый угол относительно определенной точки
RecreateRgn                  восстанавливает регион по полиполиному
PolyRotate                   Вращает полиполином на определённый угол относительно определенной точки
GetInner                     Сдвигает границу полиполинома внутрь для четного вложенного полинома и вне для нечетного
                             на заданную величину(может быть по отдельность отключено)
GetLDppoly                   Расчитывает распределение света/тени между двумя полиполиномами
RgnToPoly                    переводит регион в полиполиномное представление
SimplifyPolys                Упрощает полиполином ( удаляет повторяющиеся точки и точки лежащие на одной линии)
